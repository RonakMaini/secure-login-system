# Secure Login System

This is a mini project I developed using Flask to understand basic concepts of cybersecurity and web authentication.

The main idea of this project is to create a login system that is more secure than normal login forms by adding OTP verification and role-based access.

## Features

- User registration and login
- OTP verification after login (2-step authentication)
- Role-based access (Admin and User)
- Password stored using hashing (SHA-256)
- Session handling
- SQLite database used for storing user data

## Technologies Used

- Python (Flask)
- HTML and CSS
- SQLite

## How to Run

1. Install Flask:
   pip install flask

2. Run the project:
   python app.py

3. Open browser and go to:
   http://127.0.0.1:5000

## What I Learned

- How authentication systems work
- Basics of multi-factor authentication (OTP)
- Importance of password security using hashing
- How to connect Flask with a database
- Session management in web applications
